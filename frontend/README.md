# ProviderAgent Frontend

This is the React-based frontend for the ProviderAgent application. It connects to the existing Streamlit backend via iframe and mock API calls.

## Tech Stack
- React + Vite
- TailwindCSS
- Framer Motion
- Lucide React

## Setup

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```
   The app will run at `http://localhost:5173`.

3. **Build for Production**
   ```bash
   npm run build
   ```
